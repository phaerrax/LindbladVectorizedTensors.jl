Pseudomode operators

LindbladVectorizedTensors.dissipator_loss — Function
dissipator_loss(n::Int)

Return an OpSum object representing a dissipator operator (as in a GKSL equation) on site n, associated to the jump operator A (the annihilation operator). The OpNames A and Adag must be defined for the site type in use.

source
LindbladVectorizedTensors.dissipator_gain — Function
dissipator_gain(n::Int)

Return an OpSum object representing a dissipator operator (as in a GKSL equation) on site n, associated to the jump operator Adag (the creation operator). The OpNames A and Adag must be defined for the site type in use.

source
LindbladVectorizedTensors.dissipator — Function
dissipator(n::Int, frequency::Real, temperature::Real)

Return an OpSum object which represents the dissipator in the GKSL equation for given frequency and temperature, i.e. with dissipation coefficient 1/(exp(freq/temp) - 1).

source
